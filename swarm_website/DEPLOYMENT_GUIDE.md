## 📦 **Quick Start**

### **Backend** (Python FastAPI):
```bash
cd swarm_website/backend
pip install -r requirements.txt
python main.py
# API runs on http://localhost:8000
```

### **Frontend** (React + Vite):
```bash
cd swarm_website/frontend
npm install
npm run dev
# Frontend runs on http://localhost:5173
```
